# My build of dwm

dwm with a few patches added to it:

- Fullscreen with `Alt-Shift-F`
- Cycle layouts with `Alt-Ctrl-,` and `Alt-Ctrl-.`
- Hide vacent tags
- Scratchpad with `Alt-``
- Stacker
- Clickable modules with dwmblocks
- Sticky windows across tags with `Alt-s`
- Window swallowing
- Vanity gaps

## Install

```
cd dwm
sudo make install
```
